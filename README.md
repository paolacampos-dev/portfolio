🌐 **Portfolio Website**
This is my personal portfolio, built to showcase my work as a web designer and junior front-end developer. It highlights a selection of projects that reflect both my technical skills and my design approach.

✨ **About the Project**
The portfolio is designed with a clean, minimal aesthetic, focusing on clarity, usability, and subtle visual details. It presents a curated selection of projects, including:

    A full-stack application built with Next.js and PostgreSQL
    Interactive and UI-focused front-end projects
    A custom WordPress website built using Gutenberg and CSS

    The goal is to demonstrate both development capability and real-world application, particularly in preparation for freelance work.

🛠️ **Tech Stack**
Next.js (App Router)
React
Tailwind CSS
JavaScript (ES6+)
WordPress (Gutenberg + custom CSS)

🧩 **Features**
Data-driven project gallery using .map()
Reusable components and clean project structure
Responsive design across devices
Accessible elements (alt text, aria-labels)
Subtle UI interactions (hover effects, transitions)

🗂️ **Project Structure**
/app → pages and routing (Next.js App Router)
/components → reusable UI components
/data → structured project data
/public → images and assets

Refining UI with subtle interactions and spacing
